# Dash: SiteOrigin Widget Bundle

Requires 'Page Builder by SiteOrigin' and 'SiteOrigin Widgets Bundle' to be installed.

This widget includes

  - accordion
  - card
  - carousel
  - table
  - video background
  - youtube embed
